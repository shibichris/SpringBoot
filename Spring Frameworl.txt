Intro to Spring

Spring is an open-source lightweight framework , main technologies used 
by Spring Framework for developing enterprise applications


POJO
Dependency Injection DI
Java enterprise edition specification

Spring Framework

Relies heavily on Inversion of Control
Build using POJO - plain old java object
Spring works with java Standard edition
POJO+ configuration Metadata = Spring container
A Spring bean is a basic building block that is 
managed by Spring Framework
Spring is responsible for creating and destroying beans
Providing dependencies of the bean which could be 
other beans or configuration properties
Intercepting Bean method calls to include 
additional framework features
